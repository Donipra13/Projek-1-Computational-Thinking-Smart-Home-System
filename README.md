Smart Lighting System :
Subsistem ini berfungsi untuk mengatur kondisi pencahayaan di setiap ruangan rumah. Pengguna dapat menyalakan atau mematikan lampu secara manual, menyalakan seluruh lampu sekaligus, maupun mengaktifkan mode malam yang membuat pencahayaan menjadi lebih redup. Setiap ruangan memiliki pengaturan sendiri yang meliputi status lampu (ON/OFF) dan tingkat kecerahan.

Smart Security System :
Subsistem ini berperan sebagai sistem keamanan rumah yang menggunakan sandi digital. Pengguna dapat membuka atau mengunci pintu menggunakan kata sandi tertentu, serta memiliki akses untuk mengganti atau menambah kata sandi baru jika diperlukan. Sistem ini mensimulasikan kondisi pintu yang dapat berpindah antara status terkunci dan terbuka sesuai input pengguna.

 
Smart Thermostat System :
Subsistem ini berfungsi untuk mengatur suhu ruangan secara otomatis. Ketika suhu lingkungan melebihi batas yang telah ditentukan, sistem akan menyalakan pendingin udara (AC) secara otomatis, dan mematikannya kembali ketika suhu sudah kembali stabil. Fitur ini menggambarkan prinsip otomatisasi berdasarkan kondisi lingkungan yang berubah.
